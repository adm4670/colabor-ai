Quickstart Guide

Instale, configure e execute em 5 minutos

1. Pre-requisitos

1
Node.js 20+

Download em nodejs.org

2
npm 9+

Incluso com Node.js

3
OpenAI API Key

Ou qualquer provider OpenAI-compativel

2. Instalacao

    # Clone o repositorio
    git clone https://github.com/colabor-ai/colabor-ai.git
    cd colabor-ai
    
    # Instale as dependencias
    npm install
    
    # Configure as variaveis de ambiente
    cp .env.example .env

3. Configuracao (.env)

    # LLM Provider (OpenAI-compatible)
    LLM_API_KEY=sua-chave-aqui
    LLM_BASE_URL=https://api.deepseek.com
    LLM_MODEL=deepseek-v4-flash
    
    # Telegram (opcional)
    TELEGRAM_BOT_TOKEN=seu-token-aqui
    
    # Ambiente
    NODE_ENV=development
    LOG_LEVEL=info

4. Executar

    # Modo desenvolvimento (Telegram)
    npm run dev
    
    # Rodar testes
    npm test
    
    # Build para producao
    npm run build

5. Exemplo: Hello World com agente

    import { Orchestrator } from './core/orchestrator/orchestrator';
    import { AgentRegistry } from './core/agents/agent-registry';
    
    const registry = new AgentRegistry();
    const orchestrator = new Orchestrator(registry);
    
    const result = await orchestrator.run({
        input: "Ola, mundo! Quem e voce?",
        sessionId: "demo-001",
    });
    console.log(result); // Resposta do agente

    import { Orchestrator } from './orchestrator/orchestrator';
    const registry = new AgentRegistry();
    registry.register(plannerAgent);
    registry.register(browserAgent);
    registry.register(pythonAgent);
    registry.register(answerAgent);
    const orchestrator = new Orchestrator(registry);
    const result = await orchestrator.run({
        input: "Pesquise o clima em SP",
        sessionId: "session_123",
    });
    eventStream.emit({ type: "response", content: result });
