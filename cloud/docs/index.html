🔗 API

Endpoints

As portas de entrada do sistema. Onde o mundo externo se conecta.

POST /auth/login

Faz login com API key e recebe um token JWT. Como uma chave de hotel — so abre a porta certa.

POST /auth/verify

Verifica se um token ainda e valido. "Minha chave ainda funciona?"

POST /auth/refresh

Renova o token antes de expirar. Como renovar o aluguel da chave.

POST /chat/message

Envia uma mensagem e recebe a resposta completa. Modo pergunta-resposta tradicional.

WS ws://localhost:3001/ws?token=...&sessionId=...

Conexao em tempo real. As respostas chegam palavra por palavra enquanto sao geradas. E por aqui que o cloud envia tool_call para o cliente executar arquivos, comandos e scripts.

Assistant

cloud

O conversador. Responde perguntas, explica conceitos, da conselhos. Aquele amigo que sempre tem algo util pra dizer.

🚀 Quickstart

Como rodar

3 passos e voce esta no ar.

1

Instale as dependencias

cd cloud && npm install
2

Configure a chave da IA

Copie o arquivo de exemplo e edite com sua chave:

cp .env.example .env

Preencha DEEPSEEK_API_KEY com sua chave.

3

Rode o servidor

npm run dev

O servidor inicia em http://localhost:3001

🔧 Comandos uteis

Rodar em desenvolvimento npm run dev
Compilar TypeScript npm run build
Rodar testes (42 testes) npm test
Verificar qualidade do codigo npm run lint
Formatar codigo npm run format
