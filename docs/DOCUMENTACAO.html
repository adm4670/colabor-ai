📖 Storytelling

Como um triciclo virou nave espacial

Era uma vez um agente chamado colabor-ai. Ele funcionava — recebia uma pergunta, o Planner decidia qual agente usar, o agente executava, o Reflector avaliava. Simples, direto, funcional. Mas... ingênuo.

Se alguém pedisse "analise 5 módulos e me dê um relatório consolidado", ele processava um por um, em sequência. Se a tarefa tivesse 10 etapas, na quinta ele já esquecia o contexto. Se demorasse 30 minutos, o usuário ficava esperando. Era um triciclo — funcionava, mas não ia longe.

Foi quando surgiu a oportunidade de estudar o claude-code — o CLI agent da Anthropic, com ~500.000 linhas de TypeScript. Ali estavam as respostas: sub-agentes que spawnam em paralelo, planejamento persistente em arquivo, background tasks que rodam enquanto você dorme, hooks para extensibilidade, compactação inteligente de contexto, extração automática de memórias...

Em três ondas de implementação, o colabor-ai foi transformado. Hoje ele é capaz de decompor tarefas complexas em sub-agentes paralelos, manter planos multi-step com re-planejamento dinâmico, agendar tarefas recorrentes, consolidar memórias automaticamente, buscar na web de verdade, e muito mais. De triciclo a nave espacial — em uma única sessão. 🚀

🎯 Melhorias

Os 3 Níveis de Evolução

🔴

Nível 1 — Fundação Multi-Agente

Branch feat/melhorias-agenticas · Commit 7555a04 · +2.354 linhas

🤖 Sub-agentes

AgentTool + SubAgentRunner permitem spawnar agentes filhos em paralelo. Inspirado no AgentTool do claude-code.

Arquivos: core/tools/agentTool.ts, core/agent/sub-agent-runner.ts

📋 Planejamento Persistente

PlanManager cria/atualiza .colabor-ai/plan.md com steps, status, dependências e aprendizados.

Arquivo: core/plan/plan-manager.ts

⚡ Background Tasks

BackgroundTaskManager executa tarefas assíncronas sem bloquear a conversa principal.

Arquivos: core/tasks/background-task-manager.ts, core/tools/taskCreateTool.ts

💤 DreamTask

Consolidação periódica de memória — extrai padrões das notas diárias e atualiza MEMORY.md.

Arquivo: core/tasks/dream-task.ts

🟠

Nível 2 — Inteligência & Extensibilidade

Branch feat/nivel-2-melhorias · Commit 52bd51d · +910 linhas

🔢 TokenCounter

Contagem real de tokens com tiktoken (encoding cl100k_base). Fallback para chars/4.

Arquivo: core/context/token-counter.ts

🪝 HookManager

7 eventos (before/after planner, agent, response, on_error). Hooks com prioridade e execução assíncrona.

Arquivo: core/hooks/hook-system.ts

🧠 MemoryExtractor

Extração com frontmatter YAML + heurística. Detecta fatos, decisões, preferências, aprendizados.

Arquivo: core/memory/memory-extractor.ts

🟡

Nível 3 — Autonomia & Refinamento

Branch feat/nivel-3-refinamentos · Commit 5880f39 · +1.596 linhas

📝 TodoWrite

CRUD de TODOs persistente. Agente anota tarefas pendentes durante a execução.

Arquivo: core/tools/TodoWriteTool.ts

🔍 ToolSearch

Busca semântica TF-IDF entre 12+ tools. Agente descobre ferramentas por linguagem natural.

Arquivo: core/tools/tool-search.ts

🔄 Feedback Loop

Reflector retorna retryPrompt, confidence, suggestedAgent. Orquestrador re-rotaciona automaticamente.

Arquivo: core/agents/reflector.agent.ts

🔒 Permissions

5 níveis (read_only → full). Confirmação para operações críticas via callback.

Arquivo: core/permissions/permission-system.ts

🌐 Web Search

DuckDuckGo Instant Answer API. Busca web real, gratuita, sem API key.

Arquivo: core/tools/WebSearchTool.ts

⏰ Scheduling

node-cron + ScheduleTaskTool. Tarefas recorrentes com expressões cron.

Arquivos: core/scheduler/scheduler.ts, core/tools/ScheduleTaskTool.ts

✅

Pontos Fortes

▸ Arquitetura modular limpa (core/cloud/client) com separação clara de responsabilidades
▸ Sub-agentes com execução paralela — tarefas complexas decompostas automaticamente
▸ Plano persistente com re-planejamento dinâmico em .colabor-ai/plan.md
▸ ContextEngine com sumarização LLM e contagem real de tokens via tiktoken
▸ MemoryEngine com busca semântica + extração automática com frontmatter YAML
▸ Skills system com frontmatter — agentes carregam conhecimento sob demanda
▸ HookManager com 7 eventos para extensibilidade sem modificar o core
▸ Permission system com 5 níveis e confirmação para operações críticas
▸ 12+ ferramentas reais: web search, cron, TODOs, sub-agentes, e mais
▸ Build com zero erros em todas as 3 branches
⚠️

Pontos Fracos e Melhorias Futuras

▸ Ferramentas não integradas: email, git, banco de dados
▸ Sem interface web administrativa / dashboard
▸ Cobertura de testes insuficiente (unit/e2e)
▸ Dependência de Node.js — não roda em browser
▸ Ausência de monitoramento e observabilidade (logs, métricas)
▸ Configuração manual de variáveis de ambiente (.env)

🔮 Sugestões para próximas iterações:

🧪 Testes automatizados (Jest/Vitest) e CI/CD (GitHub Actions)
📊 Dashboard web para monitoramento de agentes e tarefas
🔌 Suporte a MCP (Model Context Protocol) para ferramentas externas
📝 Agentes definidos via arquivos Markdown (como claude-code)
🔒 Sandbox para execução segura de código Python/Shell
🌍 Suporte a múltiplos provedores LLM (OpenAI, Anthropic, Groq)
